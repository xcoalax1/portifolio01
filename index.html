Breve história do HTML5

Era o início dos anos 90, e a World Wide Web acabara de ser criada por Tim Berners-Lee. A web ainda era um lugar simples, com poucas páginas e uma interface de texto básica. Mas havia um problema: as páginas estavam se tornando cada vez mais complexas, com imagens, tabelas e links que eram difíceis de ler e navegar.

Foi nesse contexto que o HTML surgiu. HTML significa "Hypertext Markup Language" - em português, "Linguagem de Marcação de Hipertexto". A ideia por trás do HTML era simples: criar uma linguagem que permitisse aos desenvolvedores criar páginas da web mais complexas, com elementos gráficos e interativos, sem que isso comprometesse a experiência do usuário.

O que mudou no HTML5?

As principais mudanças que a nova versão proporcionará aos usuários são: Melhor tratamento de exceção, mais tags para substituir scripts, independência de plataforma e redução da necessidade de plugins externos
